## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Route-Box/route_box_web.git
   ```
2. Install packages
   ```sh
   bun install
   ```
3. Enter your environment variable in `.env`
   ```js
   VITE_API_URL = 'YOUR API_URL';
   VITE_API_KEY = 'YOUR API KEY';
   VITE_APP_BUILD_ENV = 'YOUR APP BUILD ENV';
   ```
